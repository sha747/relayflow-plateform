# relayflow-plateform

## Description
RelayFlow est une plateforme de gestion de flux logistiques orientée événements.

## Tech stack
- Java 17 / Spring Boot
- Angular
- PostgreSQL
- Messaging (RabbitMQ ou Kafka)
- Docker

## Project management
Le projet est développé en solo en simulant un environnement professionnel :
- Backlog et sprints via GitHub Projects
- Issues avec critères d’acceptation
- Branches par feature et Pull Requests
